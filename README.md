# SavoryBase

A React + TypeScript + Tailwind CMS starter focused on food website data input.

## Features

- Responsive food-item editor for desktop and mobile
- Drag-and-drop style upload area with live thumbnail framing
- Basic crop controls using zoom and image positioning
- Separate fields for food name, description, price, and category tags
- Active or inactive inventory visibility toggle
- Bulk admin actions for multi-select item updates
- Clean light gray visual theme

## Run locally

```bash
npm install
npm run dev
```
